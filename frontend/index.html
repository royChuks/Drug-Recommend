Algorithm Information
  • Logistic Regression (LR): Fast, interpretable linear model. Good baseline for classification tasks.
  • Naïve Bayes (NB): Probabilistic classifier, very fast on text, good baseline for symptoms/diseases.
  • SVM: Finds optimal separation boundaries. Effective for high-dimensional classification.
  • Random Forest (RF): Ensemble of decision trees, robust, handles multiple features well.
  • XGBoost (XGB): State-of-the-art for tabular data, gradient boosting, competition-winner.
